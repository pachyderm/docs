### Synopsis

Start a local pachd server to analyze a debug dump.

```
pachctl debug analyze <file> [flags]
```

### Options

```
  -h, --help       help for analyze
  -p, --port int   launch a debug server on the given port. If unset, choose a free port automatically
```
