## pachctl stop job

Stop a job.

### Synopsis

Stop a job.  The job will be stopped immediately.

```
pachctl stop job <pipeline>@<job> [flags]
```

### Options

```
  -h, --help             help for job
      --project string   Project containing the job (default "joins")
```
